This small application is meant to debug
Modbus communications handled by Yoctopuce
serial communication interfaces, such as 
the Yocto-RS485, the Yocto-Serial and
the Yocto-RS232 described here:

www.yoctopuce.com/EN/products/category/usb-electrical-interfaces

A precompiled version is present in the
folder bin\release 

More details about this applications can 
be found on
www.yoctopuce.com/EN/article/a-debug-tool-for-modbus
